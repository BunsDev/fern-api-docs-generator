Fern uses [Shiki](https://shiki.matsu.io/) to do syntax highlighting. This means that you can use any of the [supported languages](https://shiki.matsu.io/languages) to write beautifully highlighted code snippets.

Code snippets are written using the [Markdown fenced code blocks](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks) along with optional [syntax highlighting](https://www.markdownguide.org/extended-syntax/#syntax-highlighting).

Fern also supports enhanced syntax highlighting features such as [rendering titles](#titles) and [line highlighting](#line-highlighting).

## Basic Syntax

To create a code block, you need to wrap your code in three backticks. You can also specify the language for syntax highlighting after the opening backticks.

````mdx
```bash
npm install -g fern-api
```
````

This renders as:

```bash
npm install -g fern-api
```

## Titles

You can add a title to your code block by adding a title after the language identifier. The title must be wrapped in quotes.

````mdx
```bash title="Install the Fern CLI tool globally"
npm install -g fern-api
```
````

This renders as:

```bash title="Install the Fern CLI tool globally"
npm install -g fern-api
```

## Line Highlighting

You can highlight specific lines in your code block by placing a numeric range inside `{}` after the language identifier. The range is inclusive, so `2-4` will highlight lines 2, 3, and 4.

````markdown
```javascript {2-4}
console.log("Line 1");
console.log("Line 2");
console.log("Line 3");
console.log("Line 4");
console.log("Line 5");
```
````

This will render as:

```javascript {2-4}
console.log("Line 1");
console.log("Line 2");
console.log("Line 3");
console.log("Line 4");
console.log("Line 5");
```

The numeric range can be a single number, a comma-separated list of numbers, or ranges. For example, `{1,3,5-7}` will highlight lines 1, 3, 5, 6, and 7.

## Line Focusing

Instead of highlighting lines, you can focus on specific lines by add a `focus` attribute after the language identifier. The `focus` attribute works the same way as the `highlight` attribute.

````markdown
```javascript focus={2-4}
console.log("Line 1");
console.log("Line 2");
console.log("Line 3");
console.log("Line 4");
console.log("Line 5");
```
````

This will render as:

```javascript focus={2-4}
console.log("Line 1");
console.log("Line 2");
console.log("Line 3");
console.log("Line 4");
console.log("Line 5");
```

## Controlling the height of the code block

You can control the max height of the code block by adding a `maxLines` attribute after the language identifier. The `maxLines` attribute should be a number representing the maximum number of lines to display. By default, the code block will display up to 20 lines.

````markdown
```python maxLines=10
def is_prime(num):
    """Check if a number is prime."""
    if num <= 1:
        return False
    for i in range(2, num):
        if num % i == 0:
            return False
    return True

start = 10
end = 50

print(f"Prime numbers between {start} and {end} are:")

prime_numbers = []

for num in range(start, end+1):
    if is_prime(num):
        prime_numbers.append(num)

for prime in prime_numbers:
    print(prime)
```
````

```python maxLines=10
def is_prime(num):

```

<Tip>
  To disable the default 20 lines limit, you can set `maxLines` to `0`, or add the `fullHeight` attribute after the
  language identifier.
</Tip>

## Combining Attributes

You can combine the `title`, `highlight`, `focus`, and `maxLines` attributes to create a code block with a title, highlighted lines, and a maximum height.

````markdown
```javascript title="Hello, World!" {2-4} maxLines=5
console.log("Line 1");
console.log("Line 2");
console.log("Line 3");
console.log("Line 4");
console.log("Line 5");
console.log("Line 6");
console.log("Line 7");
console.log("Line 8");
console.log("Line 9");
console.log("Line 10");
```
````
