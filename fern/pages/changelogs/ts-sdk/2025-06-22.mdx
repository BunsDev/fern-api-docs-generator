## 1.9.0
**`(feat):`** Add `formDataSupport` configuration which lets you choose between `Node16` and `Node18`.
The default is `Node16`. If you choose `Node18`, the `form-data` dependency will be removed.


**`(feat):`** Removed the `formdata-node` and `form-data-encoder` dependencies, instead use the native `FormData` in Node 18 and up.
