---
title: Integrating npm
---

To authenticate and integrate npm within your TypeScript/JavaScript SDK, you'll need to create an npm token and add it to your repository. 

<AccordionGroup>
<Accordion title='Creating an npm token'>
<Steps>
### Log into [npm](https://www.npmjs.com/)
### Click on your profile picture and navigate to the **Access Tokens** page
### Select **Generate New Token** > **Classic Token**
### Name token and select type **Automation**
### Generate token
<Note title='Save your token'>
Be sure to save the generated token - it won't be displayed after you leave the page. 
</Note>
</Steps>
</Accordion>

<Accordion title='Adding tokens to your GitHub repository'>
<Steps>
### Open your Fern repository
### Navigate to **Settings**
### Under the Security section, open **Secrets and variables** > **Actions**
### Add **New repository secret**
### Name secret, add token, and click **Add secret**
</Steps>
</Accordion>
</AccordionGroup>
