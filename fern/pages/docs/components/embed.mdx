The Embed component allows you to include local assets directly in your documentation. This is useful for displaying PDFs, images, videos, and other media files.

## Basic Usage

Use the `embed` tag with a `src` attribute pointing to your local asset:

```jsx
<embed src="./path/to/asset.pdf" type="application/pdf" />
```

## Configuration

The Embed component supports standard HTML embed attributes:

- `src` - Path to your local asset (required)
- `type` - MIME type of the asset (recommended)
- `width` - Width of the embedded content
- `height` - Height of the embedded content

## Common MIME Types

Here are some common MIME types you might use with the Embed component:

- PDF: `application/pdf`
- MP4 Video: `video/mp4`
- WebM Video: `video/webm`
- SVG Image: `image/svg+xml`
- PNG Image: `image/png`
- JPEG Image: `image/jpeg`

<Note>
    The supported file types and behavior may vary depending on the browser and the type of content being embedded.
</Note>
