---
title: 'Aside'
excerpt: Push any content inside the Aside component to the right of the page in a sticky container
icon: 'align-right'
---

<CodeBlock title="Markdown">
````jsx
<Aside>
  
  ```ts helloWorld.ts
  export function helloWorld() {
    console.log("Hello World!");
  }
  ```

  The code snippet shows how to print `Hello World`. 
</Aside>
````
</CodeBlock>

Any markdown content inside of the `Aside` component will be 
rendered to the right of the page. This pattern is often used to 
render code samples
